# API использования отчётов

## Интерфейс IReportsInfrastructureRepository

### Основной интерфейс

```csharp
public interface IReportsInfrastructureRepository : IInfrastructureRepository
{
    Task<List<ReportInfo>> GetAvailableReportsAsync(string schemaName);
    Task<DataTable> ExecuteReportAsync(ReportInfo report);
    Task RefreshMaterializedViewAsync(ReportInfo report);
}
```

## Методы

### 1. GetAvailableReportsAsync

**Назначение:** Получить список всех доступных отчётов из БД

**Сигнатура:**
```csharp
public async Task<List<ReportInfo>> GetAvailableReportsAsync(string schemaName)
```

**Параметры:**
- `schemaName` (string) — Название схемы БД
  - PostgreSQL: `"public"`, `"reports"` и т.д.
  - SQLite: `"main"` (единственный вариант)

**Возвращает:**
- `Task<List<ReportInfo>>` — Список всех отчётов со следующими свойствами:
  - `Name` — Имя отчёта
  - `Schema` — Схема
  - `Type` — Тип ("View", "MaterializedView", "Function")
  - `Description` — Описание из БД
  - `Parameters` — Параметры (для функций)
  - `AttachDatabases` — Информация об ATTACH (для SQLite)

**Примеры:**

```csharp
// PostgreSQL
var reports = await reportsRepository.GetAvailableReportsAsync("public");
// Результат: List<ReportInfo> с View, MaterializedView, Function

// SQLite
var reports = await reportsRepository.GetAvailableReportsAsync("main");
// Результат: List<ReportInfo> с View и MaterializedView
```

**Что происходит внутри:**

PostgreSQL:
1. Запрашивает `information_schema.views` для View
2. Запрашивает `pg_matviews` для MaterializedView
3. Запрашивает `pg_proc` для Function с параметрами

SQLite:
1. Запрашивает `sqlite_master` где type='view'
2. Запрашивает `sqlite_master` где type='table'
3. Загружает метаданные из `report_metadata` для ATTACH информации

**Обработка ошибок:**
```csharp
try
{
    var reports = await reportsRepository.GetAvailableReportsAsync("public");
}
catch (InvalidOperationException ex) when (ex.Message.Contains("schema"))
{
    // Схема не существует
}
catch (Exception ex)
{
    // Другие ошибки подключения
}
```

### 2. ExecuteReportAsync

**Назначение:** Выполнить отчёт и получить результаты

**Сигнатура:**
```csharp
public async Task<DataTable> ExecuteReportAsync(ReportInfo report)
```

**Параметры:**
- `report` (ReportInfo) — Информация об отчёте
  - `Name` — Название (обязателен)
  - `Schema` — Схема (для PostgreSQL обязателен, для SQLite = "main")
  - `Type` — Тип (обязателен)
  - `Parameters` — Параметры функции (для Function)
  - `AttachDatabases` — Внешние БД (для SQLite)

**Возвращает:**
- `Task<DataTable>` — Результаты запроса с колонками отчёта

**Примеры:**

```csharp
// Простой View
var reportInfo = new ReportInfo
{
    Name = "report_sales",
    Schema = "public",  // PostgreSQL
    Type = "View"
};
var data = await reportsRepository.ExecuteReportAsync(reportInfo);

// MaterializedView
var reportInfo = new ReportInfo
{
    Name = "report_summary",
    Type = "MaterializedView",
    Schema = "main"  // SQLite
};
var data = await reportsRepository.ExecuteReportAsync(reportInfo);

// Функция с параметрами
var reportInfo = new ReportInfo
{
    Name = "get_sales_by_period",
    Schema = "public",
    Type = "Function",
    Parameters = new List<ReportParameter>
    {
        new() { Name = "p_start_date", Value = DateTime.Now.AddDays(-30) },
        new() { Name = "p_end_date", Value = DateTime.Now },
        new() { Name = "p_region", Value = "North" }
    }
};
var data = await reportsRepository.ExecuteReportAsync(reportInfo);

// С внешней БД (SQLite)
var reportInfo = new ReportInfo
{
    Name = "report_merged",
    Type = "MaterializedView",
    Schema = "main",
    AttachDatabases = "external=D:\\data\\external.db"
};
var data = await reportsRepository.ExecuteReportAsync(reportInfo);
```

**Обработка результатов:**

```csharp
var data = await reportsRepository.ExecuteReportAsync(reportInfo);

// Перебор строк
foreach (DataRow row in data.Rows)
{
    var id = row["id"];
    var name = row["name"];
    var amount = (decimal)row["amount"];
}

// Фильтрация
var filteredRows = data.Select("amount > 1000");

// Сортировка
var dataView = new DataView(data);
dataView.Sort = "amount DESC";

// Привязка к UI
dataGrid.ItemsSource = data.DefaultView;
```

**Обработка ошибок:**

```csharp
try
{
    var data = await reportsRepository.ExecuteReportAsync(reportInfo);
}
catch (ArgumentException ex) when (ex.Message.Contains("not found"))
{
    MessageBox.Show($"Отчёт {reportInfo.Name} не найден");
}
catch (ArgumentException ex) when (ex.Message.Contains("parameter"))
{
    MessageBox.Show("Не все параметры заполнены");
}
catch (InvalidOperationException ex) when (ex.Message.Contains("ATTACH"))
{
    MessageBox.Show($"Ошибка подключения внешней БД: {ex.Message}");
}
catch (Exception ex)
{
    _logger.LogError(ex, "Ошибка выполнения отчёта");
}
```

### 3. RefreshMaterializedViewAsync

**Назначение:** Обновить материализованное представление

**Сигнатура:**
```csharp
public async Task RefreshMaterializedViewAsync(ReportInfo report)
```

**Параметры:**
- `report` (ReportInfo) — Информация об отчёте
  - `Name` — Название (обязателен)
  - `Type` — Должен быть "MaterializedView"
  - `Schema` — Схема (для PostgreSQL обязателен)
  - `AttachDatabases` — Внешние БД (для SQLite)

**Возвращает:**
- `Task` — Завершается после успешного обновления

**Примеры:**

```csharp
// PostgreSQL
var reportInfo = new ReportInfo
{
    Name = "report_summary",
    Schema = "public",
    Type = "MaterializedView"
};
await reportsRepository.RefreshMaterializedViewAsync(reportInfo);

// SQLite
var reportInfo = new ReportInfo
{
    Name = "report_merged",
    Type = "MaterializedView",
    Schema = "main",
    AttachDatabases = "external=D:\\data\\external.db"
};
await reportsRepository.RefreshMaterializedViewAsync(reportInfo);
```

**Что происходит внутри:**

PostgreSQL:
1. Проверяет, что это MaterializedView
2. Проверяет наличие уникального индекса
3. Выполняет `REFRESH MATERIALIZED VIEW [CONCURRENTLY]`

SQLite:
1. Проверяет, что это MaterializedView
2. Загружает метаданные об ATTACH
3. Подключает внешние БД
4. Получает исходный SQL из `sqlite_master`
5. Удаляет старую таблицу
6. Создаёт новую таблицу
7. Сохраняет метаданные

**Обработка ошибок:**

```csharp
try
{
    await reportsRepository.RefreshMaterializedViewAsync(reportInfo);
}
catch (InvalidOperationException ex) when (ex.Message.Contains("View is not standalone"))
{
    MessageBox.Show("Нужен уникальный индекс для параллельного обновления");
}
catch (FileNotFoundException ex)
{
    MessageBox.Show($"Внешняя БД не найдена: {ex.Message}");
}
catch (Exception ex)
{
    _logger.LogError(ex, "Ошибка обновления отчёта");
}
```

## Модели данных

### ReportInfo

```csharp
public class ReportInfo
{
    /// <summary>Название отчёта/представления/функции</summary>
    public string Name { get; set; }
    
    /// <summary>Схема БД (public, main и т.д.)</summary>
    public string Schema { get; set; }
    
    /// <summary>Тип: View, MaterializedView, Function</summary>
    public string Type { get; set; }
    
    /// <summary>Описание из комментариев БД</summary>
    public string Description { get; set; }
    
    /// <summary>Параметры функции</summary>
    public List<ReportParameter> Parameters { get; set; } = new();
    
    /// <summary>Информация об ATTACH БД (SQLite)</summary>
    public string AttachDatabases { get; set; }
}
```

### ReportParameter

```csharp
public class ReportParameter
{
    /// <summary>Название параметра</summary>
    public string Name { get; set; }
    
    /// <summary>Значение параметра</summary>
    public object Value { get; set; }
    
    /// <summary>Тип .NET</summary>
    public Type Type { get; set; }
    
    /// <summary>Тип БД</summary>
    public string DbType { get; set; }
    
    /// <summary>Обязателен ли параметр</summary>
    public bool IsRequired { get; set; }
    
    /// <summary>Описание</summary>
    public string Description { get; set; }
}
```

## ReportService

Сервис более высокого уровня, работающий с ReportService:

```csharp
public interface IReportService
{
    Task<List<ReportInfo>> GetReportsListAsync();
    Task<DataTable> GetReportDataAsync(ReportInfo report);
    Task RefreshReportAsync(ReportInfo report);
}
```

### Использование через ReportService

```csharp
public class MyViewModel : ViewModelBase
{
    private readonly IReportService _reportService;
    
    public async Task LoadReports()
    {
        // Получить список отчётов (фильтруется по сконфигурированной БД)
        var reports = await _reportService.GetReportsListAsync();
        
        // Выполнить отчёт
        var data = await _reportService.GetReportDataAsync(reports[0]);
        
        // Обновить отчёт
        await _reportService.RefreshReportAsync(reports[0]);
    }
}
```

## Интеграция с UI (XAML)

### ViewModel

```csharp
public class ReportsControlVM : ViewModelBase
{
    private readonly IReportService _reportService;
    
    public ObservableCollection<ReportViewModel> ReportsList { get; }
    public ObservableCollection<DataRowView> ReportResults { get; }
    
    public ReportsControlVM(IReportService reportService)
    {
        _reportService = reportService;
        ReportsList = new();
        ReportResults = new();
    }
    
    [RelayCommand]
    public async Task LoadReports()
    {
        try
        {
            var reports = await _reportService.GetReportsListAsync();
            ReportsList.Clear();
            foreach (var report in reports)
            {
                ReportsList.Add(new ReportViewModel(report));
            }
        }
        catch (Exception ex)
        {
            MessageBox.Show($"Ошибка: {ex.Message}");
        }
    }
    
    [RelayCommand]
    public async Task ExecuteReport(ReportViewModel selectedReport)
    {
        if (selectedReport == null) return;
        
        try
        {
            var reportInfo = new ReportInfo
            {
                Name = selectedReport.Name,
                Type = selectedReport.Type,
                Schema = selectedReport.Schema
            };
            
            var dataTable = await _reportService.GetReportDataAsync(reportInfo);
            ReportResults.Clear();
            foreach (DataRowView row in dataTable.DefaultView)
            {
                ReportResults.Add(row);
            }
        }
        catch (Exception ex)
        {
            MessageBox.Show($"Ошибка: {ex.Message}");
        }
    }
    
    [RelayCommand]
    public async Task RefreshReport(ReportViewModel selectedReport)
    {
        if (selectedReport?.Type != "MaterializedView") return;
        
        try
        {
            var reportInfo = new ReportInfo
            {
                Name = selectedReport.Name,
                Type = "MaterializedView",
                Schema = selectedReport.Schema
            };
            
            await _reportService.RefreshReportAsync(reportInfo);
            MessageBox.Show("Отчёт обновлён");
        }
        catch (Exception ex)
        {
            MessageBox.Show($"Ошибка: {ex.Message}");
        }
    }
}
```

### View (XAML)

```xml
<DataGrid ItemsSource="{Binding ReportsList, Mode=OneWay}"
```

## Фоновое обновление (BackgroundService)

```csharp
public class ReportRefreshBackgroundService : BackgroundService
{
    private readonly IServiceProvider _serviceProvider;
    private readonly ILogger<ReportRefreshBackgroundService> _logger;
    
    public ReportRefreshBackgroundService(
        IServiceProvider serviceProvider,
        ILogger<ReportRefreshBackgroundService> logger)
    {
        _serviceProvider = serviceProvider;
        _logger = logger;
    }
    
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            try
            {
                using var scope = _serviceProvider.CreateScope();
                var reportService = scope.ServiceProvider.GetRequiredService<IReportService>();
                
                // Получить все отчёты
                var reports = await reportService.GetReportsListAsync();
                
                // Обновить только MaterializedView
                var mvReports = reports.Where(r => r.Type == "MaterializedView").ToList();
                
                foreach (var report in mvReports)
                {
                    try
                    {
                        _logger.LogInformation($"Обновление отчёта: {report.Name}");
                        await reportService.RefreshReportAsync(report);
                    }
                    catch (Exception ex)
                    {
                        _logger.LogError(ex, $"Ошибка при обновлении {report.Name}");
                    }
                }
            }
            catch (Exception ex)
            {
                _logger.LogError(ex, "Ошибка в фоновом сервисе обновления");
            }
            
            // Обновлять каждый час
            await Task.Delay(TimeSpan.FromHours(1), stoppingToken);
        }
    }
}
```
